
# Thanolo Portfolio Site

A professional, responsive single‑page portfolio for GitHub Pages.

## Deploy on GitHub Pages
1. Create a **public** repository on GitHub.
2. Upload all files (keep `index.html` at the repo root).
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set **Source** to *Deploy from a branch*.
5. Choose the `main` branch and **/(root)** folder, then **Save**.
6. Wait 1–2 minutes. Your site will appear at `https://YOUR_USERNAME.github.io/REPO_NAME`.

## Customize
- Update colors in `assets/css/style.css` via the `:root` variables.
- Replace avatar or add a headshot at `assets/img/` and reference it in HTML.
- Replace `assets/Thanolo-Resume.pdf` with your latest PDF if desired.
